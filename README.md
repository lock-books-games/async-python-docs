# Async Programming in Python

This repository contains documentation and examples
covering asynchronous programming concepts in Python.

## Contents
- Core concepts: sync vs async, concurrency, parallelism
- CPU-bound vs IO-bound tasks
- asyncio fundamentals
- Python-specific constraints (GIL)

## Structure
- `docs/` — documentation pages
- `examples/` — runnable Python examples
- `images/` — diagrams used in documentation

## Who is this for?
- Python developers
- Backend engineers
- Readers new to async programming

## How to read
Start with `docs/index.md`.
